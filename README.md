# pdf-extractor
A simple CLI tool to extract a page range from a PDF and save it as a new file.
PDF Page Extractor | PDF 页面提取工具

A simple interactive command-line tool to extract a range of pages from a PDF file and save them as a new PDF.
一个简单的交互式命令行工具，用于从 PDF 文件中提取指定页码范围，并保存为新的 PDF 文件。

Features / 功能：

Drag-and-drop file path input / 支持拖拽文件路径输入
Auto-detects total page count / 自动读取 PDF 总页数
Custom start & end page selection / 自定义起始和结束页码
Custom output path or auto-save to source folder / 支持自定义输出路径或自动保存至原目录
Requires / 依赖： PyPDF2
